2 # SETestKCL : <i>Negotiation Study</i>
3 
4 <b>This <i> software </i> can be used to count the number of "yes" and "no" recorded during a negotiation. </b>

7 
8 It is based on Qt for the gui part.

13 ## REQUIREMENTS for compilation:
14 
15 There are 3 pieces of software that are required for the correct installation of this software:
16 
17  - [CMAKE](https://cmake.org/) with version > 3.0. Required for the Cross-Platform makefile creations
18  - [BOOST](http://www.boost.org/) with version > 1.60.0. Required for the rapid estimation of binomial coefficients. Note that only the includes are necessary. Simply download the source code. And, during cmake configuration, point the option Boost_INCLUDE_DIR to the main root directory of the downloaded folder.
19  - [QT](http://www.qt.io/) with version > 5.0. Required for the creation of the user interface.
20 
21 Once these packages have been downloaded and installed, you can download the source code of SETestKCL software and start cmake configuration:
22 
23  git clone https://github.com/ntoussaint/SETestKCL.git
24 
25  cd SETestKCL; mkdir build; ccmake ..
26 
27 Press 'c' for configuration. Simply configure the cmake to your convenience (be careful that the Boost_INCLUDE_DIR points to the correct directory). Press 'g' to generate makefiles.
28 
29 After compilation, the binary (or bundle) should be created under the 'bin' directory of the build tree. Alternatively you can 'make install' the software for more global access.
30 
31 Launch the application using the following commands:
32 
33 For Linux:
34 
35  . bin/negotiation
36 
37 For Mac:
38 
39  open -a negotiation ./bin/negotiation.app
40 
41 For Windows:
42 
43  cmd ./bin/negotiation.exe

49  - type 'y' each time the negotiator says "yes"

51  - type 'n' each time the negotiator says "no"
52 
53  - type 'z' to undo the last operation
54

56 
57  - type 'q' to exit
58
